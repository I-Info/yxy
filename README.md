## How to run
1. Prepare `Rust` development environment. 

2. Clone the repo
    ``` bash
    $ git clone https://github.com/I-Info/yxy.git
    ```

3. Change the working directory
    ```bash
    $ cd yxy
    ```

4. Edit the `conf.yaml` (example in `conf.example.yaml`)
    
5. Compile & Run by Cargo
    ``` bash
    $ cargo run
    ```
